Welcome to ThemisPy's documentation!
====================================

ThemisPy is a set of postprocessing tools written in Python_ for Themis_ analyses of Event Horizon Telescope (and ancillary) data sets.  It is under continual development by the Themis Development Team, so stay tuned!


.. _Python: https://www.python.org/
.. _Themis: https://github.com/PerimeterInstitute/Themis
